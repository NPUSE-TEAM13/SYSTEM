# 项目：职员信息服务管理系统
“SISMS职员信息服务管理系统”是公司管理高效化、现代化重要部分，主要从公司员工管理工作出发、涵盖了职员信息电子化管理、职员入职管理、宿舍管理、技能培训管理、奖惩管理、离职管理等方面。利用本系统可以达到日常工作的无纸化办公、并为公司职员管理工作带来便利。同时，信息的电子化管理为日后的信息统计、信息查看提供强有力的保障。
